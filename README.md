# Analise-Redes-Sociais-dos-Perfis-Socioeconomicos-dos-Beneficiarios-do-Bolsa-Familia
Este estudo aplica a metodologia de Análise de Redes Sociais (ARS) para  investigar a organização estrutural das características socioeconômicas associadas a  domicílios beneficiários e não beneficiários do Programa Bolsa Família no Brasil.
